üîß How Pizza Order Debouncing Works

Step 1: Customer starts typing
User types "p" ‚Üí Timer starts (300ms countdown)
Step 2: More typing happens
User types "e" ‚Üí Previous timer cancelled, new timer starts
Step 3: Keep typing...
User types "p-p-e-r-o-n-i" ‚Üí Timer keeps getting cancelled and restarted
Step 4: Customer pauses
300ms passes with no typing ‚Üí Kitchen check happens!
Step 5: Results delivered
Tony checks ingredients once and responds with pizza options

// üçï The Magic Debounce Function
function debounce(func, delay) {
    let timeoutId;  // üì¶ Store the timer
    
    return (...args) => {
        clearTimeout(timeoutId);  // ‚ùå Cancel previous timer
        timeoutId = setTimeout(() => {
            func.apply(null, args);  // üéØ Call the function
        }, delay);  // ‚è∞ After the delay
    };
}
